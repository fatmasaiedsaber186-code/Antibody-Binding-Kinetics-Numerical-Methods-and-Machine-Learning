# Antibody Binding Kinetics: Numerical Methods and Machine Learning

## Project Overview
This project solves the Antibody Binding Kinetics model using Finite Difference Method (FDM), Finite Element Method (FEM), and three Machine Learning models (PINN, DeepONet, and GNN). It compares accuracy, computational performance, and prediction capability.
